Проверка разницы форм отчетности.

Использование:
Первая кнопка - загрузить отчет за предыдущий период.
Вторая кнопка - загрузить отчет за текущий период.
Третья кнопка - произвести расчет.

Загружаемые отчеты должны быть в формате XLSX и должны иметь название формы в начале названия файла(f1..., f4... и др).
После нажатия кнопки "Обработать" происходит вычитание таблиц, по окончанию которого сохраняется и открывается новый файл с припиской "Разница"

Рассчитывать можно формы любых видов, с полной кастомизацией:
Файл shablons.xlsx содержит листы с параметрами форм (название листа является тем критерием, по которому проверяется имя файла)
В листах описание разделов: на каком листе они находятся и какие координаты таблиц с числами имеют.
